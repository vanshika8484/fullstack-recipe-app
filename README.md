# 🍲 Recipe Finder App

A **React Native mobile application** that helps users discover delicious recipes, watch cooking tutorials, and save their favorite recipes for later.

This app allows users to search recipes instantly, view ingredients and instructions, and learn how to cook through integrated YouTube tutorials.

## 🚀 Features

- 🔐 **User Authentication**
  - Secure **Login & Signup**
  - Personalized user experience

- 🔎 **Recipe Search**
  - Quickly search for recipes by name

- 📖 **Recipe Details**
  - Ingredients list
  - Step-by-step cooking instructions

- 🎥 **YouTube Tutorials**
  - Embedded cooking tutorials for recipes

- ❤️ **Save Recipes**
  - Users can save their favorite recipes
  - Access them anytime later

- 📱 **Clean Mobile UI**
  - Built with React Native
  - Smooth and responsive design

## 🛠 Tech Stack

**Frontend**
- React Native
- JavaScript

**Backend / Services**
- Recipe API
- YouTube API (for tutorials)

**Tools**
- Expo / React Native CLI
- Git & GitHub

## ⚙️ Installation

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

git clone https://github.com/vanshika8484/fullstack-recipe-app.git

### 2️⃣ Navigate into the project

cd mobile

### 3️⃣ Install dependencies

npm install

### 4️⃣ Start the development server

npx expo start

### 5️⃣ Run the app

- Open with **Expo Go** on your mobile device  
OR  
- Run on **Android Emulator / iOS Simulator**
